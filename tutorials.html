1) Importing excel file


Importing excel files is really straightforward, click the big browse button and browse to find your file

Eventually the top 10 values of your data will pop up in the files section




2) Importing text file


Importing text files is really straightforward,first select the text file option in type of file dropdown

then select the delimiter for your data,if not available click the 'Not here' option and

write down the delimiter manually. Eventually the top 10 values of your data with pop up in the files section




3) Choosing the options


This is again, really simple, just choose the x axis and y axis column

Then choose the aggregation method, whether you want the data to be summed, counted(frequency), averaged, or median

Then fill up the x and y axis labels and the title

Then choose the type of graph

If the x axis is categoric or dates, then the available options are:

  • Bar graph
  • Pie chart
  • Box and whisker plot
  • Time series plot (for dates or years)
  • World Map (for countries)
  • USA Map (for states)

    • If the x axis is categoric, then the available options are:

  • Scatter plot
  • Histogram

    • There you have it, all the options have been set up




    4) The colour options


    The colour options simply selects a colour for your bar depending on the option you've chosen

    This option is available in the following graphs

  • Bar graph
  • Box and whisker plot (for categorical values and dates)
  • Scatter plot
  • Histogram (for categorical values and dates)

    • Numeric value for colour shows a colourscale ,which you can change using the colourscale option, and a categorical value shows a legend assigned to arbitary colours

    You change the layout of the option by choosing the group and stack radiobuttons

    The group options puts the bars separately, whereas the stack option stacks them up (this is only available for bar graphs)




    5) The Label options


    Datum provides interactive graphs for you, this can only be implemented through labels

    Labels are all numeric values, but if you want you're x axis or the colour (written as a categorise) column as a label, then just click the checkbox available

    The first text Input is the preceding text ,which comes before the column value, and the second one is the text that comes after the certain column you have chosen

    You can also select the units whether you want it as original form, thousands, millions or billions

    You can also select a maximum of four columns as label including two extra ones, that is, the x axis label and the colour column label

    You can also choose the labels as text checkbox, once you click that all labels will be written as text, but the more numbers of bins there are the more, the text will get more smaller




    6) Additonal lines


    Datum also provides trend lines or target lines

    You can firstly choose whether you want an average, median or customised line

    Average line/Median line will be a constant line and its y axis will be set to the average/median of the data,

    Target line as label option turns that boring line, interactive, where you actually see the value

    Customised line is where you can choose a target to achieve and set it to a certain value

    Once you add target lines colour option and animation option would be disabled

    You can change the colour of the line using the colour picker




    7) Animations


    Another cool feautre is, that you can add cool animations to you graphs

    Just tick the animate button and whole lot of options will appear

    Choose the columns, it's most appropriate to choose date and years for this

    Set the frame rate, how long it'll change each frame (in milliseconds)

    Then if you want a slider you can tick the slider visibility box, then hit the play button and your animation would play

    Once you use animations the trendline option, label option and colour option would be disabled




    8) Filter the data


    You can filter your data using our epic data filter

    You can change the 'Original' dropdown to anything like number of vowels or is prime or is perfect square

    If you have numeric or date values you can use greater than, less than operators, equal to and not equal to operators

    You put multiple values in a dropdown

    And there you have it, cleansed data, with no mess
